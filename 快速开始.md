# 快速开始指南 🚀

## 您的语言学习助手已经创建完成！

这个应用可以通过拍摄物品来帮助用户学习外语。以下是快速启动步骤：

## 📱 立即运行应用

### 步骤 1: 打开项目
```bash
cd "/Users/shaomeichen/Desktop/xcode/Intelligent identification"
open "Intelligent identification.xcodeproj"
```

### 步骤 2: 配置签名
1. 在 Xcode 左侧导航器中点击项目名称（蓝色图标）
2. 选择 "Intelligent identification" target
3. 点击 "Signing & Capabilities" 标签
4. 勾选 "Automatically manage signing"
5. 在 "Team" 下拉菜单中选择您的 Apple ID

### 步骤 3: 选择设备并运行
1. 连接您的 iPhone（推荐）或选择模拟器
2. 点击 Xcode 左上角的 ▶️ 按钮（或按 ⌘R）
3. 首次运行时允许相机权限

## ✨ 已实现的功能

### 核心功能
✅ **相机拍摄** - 实时相机预览和拍照  
✅ **AI识别** - 使用Apple Vision框架识别物体

## 🔧 常见问题

### Q: 需要网络连接吗？
**A**: 不需要！所有功能都可以离线使用。

### Q: 为什么模拟器无法使用相机？
**A**: iOS模拟器不支持真实相机。请在真实设备上测试。

### Q: 如何添加更多物品？
**A**: 编辑 `TranslationService.swift` 文件，在 `translations` 字典中添加。

### Q: 识别不准确怎么办？
**A**: 
- 确保光线充足
- 物品清晰可见
- 背景简洁
- 物品占据画面主要部分

## 🚀 进阶定制

### 添加新语言
在 `TranslationService.swift` 中：
```swift
private let translations: [String: [String: String]] = [
    "apple": [
        "en": "Apple",
        "zh": "苹果",
        // 添加新语言
        "de": "Apfel"  // 德语
    ],
    // ...
]
```

### 添加新物品
```swift
"laptop": [
    "en": "Laptop",
```
