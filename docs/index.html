拍照识别一秒懂，开口就能学

AI智能识别是你的视觉语言助手：一拍即识、多语种翻译、标准发音、iCloud 同步与离线查看。把“所见”转化为“会说”。

快速开始

  1. 在设置里选择“母语”和“学习语言”。
  2. 点击首页“开始识别”，对准目标物体稳定取景。
  3. 查看顶部结果卡片：双语名称 + 简短用途提示。
  4. 点按播放，听标准发音并跟读。

功能亮点

  • 多语言：中文、英语、西语、法语、日语、韩语…
  • 智能翻译兜底：本地词典未命中时自动调用在线翻译。
  • 学习记录：自动沉淀，离线可看；iCloud 加密同步。
  • Apple 登录与邮箱登录，体验无门槛。
